# node编写获取微信公众号文章所有图片的爬虫

## 使用
### 1.安装依赖
``` bash
npm install 
```
### 2.修改index中的链接
``` js
startDown(你的链接地址); 
```
### 3.开始爬取
``` bash
npm start 
```
